протокол - интерфейс
TCP/IP - протоколы передачи данных в интернете.
TCP (Transmission Control Protocol) - протокол для управления передачи данных
IP (Internet Protocol) - протокол определяет маршруты данных в интернете
DNS (Domain Name System) - система доменных имен используется
для получения IP по имени hosts(server)
MAC-адрес (Media Access Control) - уникальный идентификатор,
присваиваемый каждой единице активного оборудования 
Мултиплексирование - передача нескольких потоков данных
с меньшей скоростью по одному каналу
физическая(прокладка кабелей) и логическая(как ходят пакеты
в физической) топологии
модель OSI
1. физический уровень - передача электрических сигналов
2. канальный адресация в пределах локальной сети МАС-адрес, пртокол Ethernet
3. Сетевой уровень IP
4. Транспортный уровень TCP(установленное соединение)
UDP(неустановленное соединение) (порты)
5. Сеансовый уровень - управление и разырв соединения между двумя хостами
поддерживает сеансы пользователей
6. Представления уровень - читабельный вид прикладного уровня (ASCII, JPEG)
7. Прикладной уровень HTTP, FTP
нельзя перескакивать между уровнями модели OSI
модель TCP/IP
1. прикладной(представления и сеансовый)
2. транспортный
3. интернет(сетевой уровень)
4. уровень сетевого доступа(канальный и физический)
